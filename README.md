# Flask Portfolio Website

This project is a simple portfolio website built using Flask.

## Features
- Home page with personal information
- Contact form
- Flask routing
- HTML and CSS styling

## Technologies Used
- Python
- Flask
- HTML
- CSS

## How to Run

1. Install Flask
pip install flask

2. Run the app
python app.py

3. Open in browser
http://127.0.0.1:5000
